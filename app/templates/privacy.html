Open History
Privacy by Design

Records stay local unless the user explicitly exports them.

ELORA stores daily records in a local SQLite file and provides export and deletion controls on the History page.

User Control

Export to CSV when needed, or permanently remove records with Clear History.

Stored

What data does ELORA store?

  • Daily records: date, age, gender, height, weight
  • Lifestyle fields: family history flag, activity level, water intake
  • Derived BMI value per record

No name, email address, student ID, address, or contact details are stored.

Location

Where is the data stored?

  • Data is stored in a local SQLite database file under the ELORA project folder.
  • The prototype does not send records to any remote server or cloud service.
Control

How can the user control their data?

  • Use Download CSV on the History page to export a copy.
  • Use Clear history on the History page to permanently delete local records.

Use in research and teaching

This prototype is intended for demonstration and educational purposes only. It does not provide medical diagnosis or treatment recommendations. Any health-related decisions should be made with a qualified clinician.
